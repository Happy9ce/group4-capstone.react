# Solar System Explorer - Milestone Project

A React-based web application that showcases planetary data through an interactive and visually appealing interface.

## Features

- **Hero Section**: Introduction to the solar system with imagery
- **Video Section**: Educational video about planetary science
- **Gallery Section**: Visual comparisons of different planets
- **Data Table**: Comprehensive planetary facts (mass, diameter, density, gravity)
- **Contact Form**: User inquiry form with validation
- **Responsive Design**: Mobile-friendly layout with media queries

## Tech Stack

- React 19
- Vite
- CSS3

## Project Structure

```
src/
├── layouts/          # Header and Footer components
├── sections/         # Page sections (Hero, Video, Gallery, Table, Form)
├── App.jsx          # Main application component
├── App.css          # Global styles
├── main.jsx         # Entry point
└── index.css        # Base styles
```
